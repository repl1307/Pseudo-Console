Pseudo Console

log( )

 
  //this method is what should be called in your scripts
  debug.log('This is sample output');
    
  //these methods will only work in the console text input 
  log('This is sample output'); 
  this.log('This is sample output');
  console.log('This is sample output');

Expected Output:

Outputs a message to the console

This method accepts any data type as a parameter.

  debug.clear();

Note: The console doesn't appear in mobile browsers.
